# Ghost Shot Coach

Ghost Shot Coach converts a measured basketball shooting-form flaw into one
plain-language correction and one actionable, source-cited drill.

The packaged capability is contract-driven: it accepts Ghost's directional
`Flaw` shape and returns its validated `CoachingResult` shape. It is designed
for agent reuse, free ERC-1155 access, and transparent citation provenance.

Asset slug: `ghost-shot-coach_finchip`

Status: package-ready, not minted. Publishing requires a FinChip `fc_key`,
registered wallet, funded chain account, and content upload/IPFS credentials.
